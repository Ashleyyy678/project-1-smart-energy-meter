# Smart Energy Meter - Web Dashboard

A fully responsive single-page website for monitoring household energy usage, powered by ESP32.

## Features

- **Live Dashboard**: Real-time monitoring of power, current, voltage, and energy usage
- **Appliance Management**: Track individual appliance voltage, current, and power consumption
- **Alerts System**: Monitor overcurrent, overvoltage, and unusual spikes
- **Insights & Analytics**: View energy usage patterns, peak times, and weekly comparisons
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## How to View

Simply open `index.html` in any modern web browser. No server required!

### Quick Start:
1. Download or clone this repository
2. Open `index.html` in your browser
3. Navigate through sections using the top navigation bar

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and data visualization (Chart.js)

## Technology Stack

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript
- Chart.js (for data visualization)

## Note

This is a front-end only demo. All data is simulated for demonstration purposes. Real ESP32 data integration can be added later.
