# How to Share This Project with Your Group

## Option 1: GitHub (Recommended for Group Projects) ‚úÖ

### Step 1: Initialize Git Repository
```bash
git init
git add .
git commit -m "Initial commit: Smart Energy Meter website"
```

### Step 2: Create GitHub Repository
1. Go to [github.com](https://github.com) and sign in
2. Click the "+" icon in top right ‚Üí "New repository"
3. Name it: `smart-energy-meter` (or any name you prefer)
4. **Don't** initialize with README (we already have one)
5. Click "Create repository"

### Step 3: Push to GitHub
```bash
git remote add origin https://github.com/YOUR_USERNAME/smart-energy-meter.git
git branch -M main
git push -u origin main
```

### Step 4: Share with Group Members
- Send them the repository URL: `https://github.com/YOUR_USERNAME/smart-energy-meter`
- They can clone it: `git clone https://github.com/YOUR_USERNAME/smart-energy-meter.git`
- Or download as ZIP from GitHub

### Bonus: Enable GitHub Pages (Free Hosting!)
1. Go to repository Settings ‚Üí Pages
2. Source: Deploy from branch ‚Üí main
3. Save
4. Your site will be live at: `https://YOUR_USERNAME.github.io/smart-energy-meter`

## Option 2: Cloud Storage (Quick & Easy)

### Google Drive / OneDrive / Dropbox
1. Zip the entire folder:
   - Select all files (index.html, styles.css, script.js, README.md)
   - Right-click ‚Üí "Send to" ‚Üí "Compressed (zipped) folder"
2. Upload the ZIP to Google Drive, OneDrive, or Dropbox
3. Share the link with your group members
4. They download and unzip to view

**Files to include:**
- index.html
- styles.css
- script.js
- README.md
- .gitignore (optional)

---

## Option 3: Email Attachment

1. Zip all files (same as Option 2)
2. Attach ZIP file to email
3. Send to group members

‚ö†Ô∏è **Note:** Some email providers have size limits (usually 25MB). Our files are tiny, so this should work fine!

## Option 4: USB Drive / Shared Folder

If working on same network or in person:
- Copy the entire folder to USB drive
- Or copy to shared network folder
- Group members copy to their computers

---

## What Group Members Need to Do:

1. **Download/Clone** the files
2. **Unzip** (if downloaded as ZIP)
3. **Open `index.html`** in any web browser
4. That's it! No server needed - works directly from files

---

## Best Option for Your Group?

- **GitHub**: Best for collaboration, version control, and free hosting
- **Cloud Storage**: Fastest for quick sharing without setup
- **Email**: Simplest if just sharing once

All options work perfectly - choose what's easiest for your team! üöÄ
