About the Project

How It Works

  • ESP32 with isolated voltage and current sensors (e.g., ACS712 + voltage sensor module)
  • Measures real-time RMS voltage, current, and computes power
  • Logs data for each circuit/appliance
  • Data is sent to this web dashboard over Wi-Fi

Project Objectives

  • Reduce energy costs
  • Detect phantom loads
  • Improve safety via overcurrent/overvoltage alerts
  • Provide clear, user-friendly visualizations
  • Enable data-driven energy management decisions
